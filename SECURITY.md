# Security Policy

## Supported Versions

Currently, only the latest version will **ever** be supported as any outdated version most likely means an update to the API.

| Version  | Supported |
|----------|-----------|
| v v4.x.x | ✅         |
| < v4.0.0 | ❌         |
| < v3.0.0 | ❌         |
| < v2.0.0 | ❌         |
| < v1.0.0 | ❌         |

## Reporting a Vulnerability

For any security vulnerability, please create a new [security issue](https://www.github.com/beanbeanjuice/cafeBot/issues) on the issues tab.
